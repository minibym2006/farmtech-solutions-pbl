# 🚜 FarmTech Solutions - Previsão de Rendimento de Safra

Bem-vindo(a) ao repositório do projeto de Machine Learning desenvolvido para a **FarmTech Solutions**. 

Este projeto faz parte da Entrega 1 (Fases 4 e 5) e tem como objetivo ajudar uma fazenda de médio porte (aproximadamente 200 hectares) a analisar e prever o rendimento das suas safras com base em condições climáticas (precipitação, umidade e temperatura) e tipos de culturas.

## 🎯 Objetivo do Projeto

Através dos dados fornecidos no arquivo `crop_yield.csv`, nosso time desenvolveu uma solução completa de Inteligência Artificial que inclui:
1. **Análise Exploratória de Dados (EDA):** Compreensão das variáveis e do impacto do clima na produtividade.
2. **Machine Learning Não Supervisionado:** Clusterização e identificação de anomalias (outliers) para descobrir tendências de rendimento.
3. **Machine Learning Supervisionado:** Construção, treinamento e avaliação de **5 modelos preditivos de regressão** para prever com precisão o rendimento das futuras plantações em toneladas por hectare.
